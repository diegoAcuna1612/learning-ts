**Concepto:** Transformar interfaces existentes sin tener que escribir nuevas desde cero. Muy útil para formularios de edición o tablas de resumen.

- **Ejercicio:**
- - Usa `Partial<Predio>` para crear una función que solo actualice algunos campos.
  - Usa `Pick<Expediente, 'id' | 'numero'>` para una tabla que solo muestra lo básico.
  - Usa `Omit<Solicitud, 'id'>` para el objeto que envías al crear una solicitud nueva (porque el ID lo genera la DB).
